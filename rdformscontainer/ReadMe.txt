How to configure
================


rdfom-bookmarklet-local.html
----------------------------
Change the path of 'includeJS.js' so that it can be accessible from your browser.



includeJS.js
------------
Change the 'dir' variable according to your directory structure so that 'ui.dialog.js' and 'rdform-container.js' can be loaded.



twitter-rdform.js
-----------------
Change the 'loadForm' variable to load any html form.
Change the 'img' path in 'contentdiv' variable according to your directory structure so that the image can be loaded.
